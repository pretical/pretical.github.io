Service

弊社が提供する5つの主要業務です。日本国内はもちろんのこと、「海外に通じる」、「海外でも通じる」を求めるクライアント様にも対応するため、日英バイリンガル（または多言語）かつ海外のデザイン・ユーザビリティの基準に沿うものを提供いたします。

  • Consulting
  • Web Development
  • UX Design
  • Translation
  • Internal IT

海外への事業展開・情報発信や、資料やウェブサイトの多言語対応、海外向けブランドコンサルティングなどを行います。商材に応じて国際市場の英語リサーチの代行なども行いますので、業界の最新のトレンドに基づいて海外展開のお手伝いをいたします。

レスポンシブ対応の新規ウェブサイトの立案、企画、デザイン、開発、運営を行います。
既存ウェブサイトのリニューアル、オンラインストアの新設や既存サイトへの連携も可能です。

開発環境　Linux / Mac / Windows
使用言語　HTML5 / CSS3 / Sass / Javascript
フレームワーク Hover / Animate / jQuery / Bootstrap / Semantic UI
CMS　Jekyll/Wordpress

毎月の維持料金が一切発生しないプランもございますのでお気軽にお問い合わせください

UXデザインはユーザエクスペリエンス、すなわち「利用者体験」に基づき、そこから得られた知見を体験向上へとつなげるという比較的新しいデザイン分野です。文化や人間活動の常識や物理的な動作に基づいた、もっとも自然で効率的なデザインの在り方を実現することでサービスの利用満足度を高めるのと同時に、ユーザの行動や意見から学習した内容をデザインに反映させるサイクルを継続的に行います。

日英の翻訳・通訳を行います。

海外事業者との契約書やNDA（秘密保持契約）、RFI/RFPといったビジネス資料からマニュアル翻訳まで自社内にて幅広く対応いたします。また、英語ウェブサイトのデザインをご依頼されるお客様には海外案件のメール連絡の英語サポートなども行っております。

パートナーを通して、特許関係や専門職の強い内容や中国語・スペイン語・フランス語といった英語以外の言語にも対応可能です。

通訳では、商談やスピーチなど、逐次通訳および同時通訳の対応が可能です。ビデオ会議を通しての海外との交渉などにも対応いたします。（事前打ち合わせが必要となる場合もございます）

社内でのデジタルツール活用のご提案をいたします。SFA/CRM、グループウェア、Eメールマーケティングツールなど、無料・有料のクラウドツールの最適な利用を進めることで経営の効率化や既存ツールの維持費の最小化を図ります。
